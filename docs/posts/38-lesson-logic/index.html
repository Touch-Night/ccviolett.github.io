每一个直言命题都由四个要素构成：量项-主项-联项-谓项

如：量项(所有) 主项(艺术家) 联项(都是) 谓项(人)

直言命题的四种形式：

  • A：所有 S 都是 P
  • E：所有 S 都不是 P
  • I：有些 S 是 P
  • O：有些 S 不是 P

直言命题的量：根据直言命题的量项，分为 全称命题 与 特称命题。

直言命题的质：根据直言命题的联项，分为 肯定命题 与 否定命题。

1.1 逻辑学是什么

  1. 以下（A）不属于逻辑系统四大定理。
  • A 不完全性定理
  • B 有效性定理

逻辑系统的四大定理：

  1. 一致性定理
  2. 有效性定理
  3. 可靠性定理
  4. 完全性定理
  1. 逻辑主义的代表人物是（D）
  • A 布劳威尔
  • B 哥德尔
  • C 希尔伯特
  • D 罗素

逻辑主义( logicism)亦称“逻辑斯提”，即“数学哲学中的逻辑主义”，主张数学是逻辑的延伸。

他们只研究概念间的纯逻辑关系，从逻辑学可推导出全部数学，全部数学可以归结为逻辑学。 其代表人物是弗雷格、罗素、怀特海。

数学四大流派：柏拉图主义、逻辑主义、形式主义、直觉主意

2.3 经验命题与必然命题

  1. 以下属于必然真命题的是（D）
  • A 儿童是人
  • B 女人是人
  • C 中国人不是中国人
  • D 中国人是中国人
  1. “男人是人”是（AC）。
  • A 真命题
  • B 假命题
  • C 经验命题
  • D 必然命题
  1. （x）经验命题是指需要根据直接的观察经验来判断真假的命题。

我们把需要根据观察经验才能判定其真假的命题称为经验命题 （empirical proposition）。

例如，“贵州省在中国的西南部”这个命题要么为真要么为假。要判定其真假，只要某人去过贵州并且去过西南，他就能判定这个命题是真的。但是，这种经验并不必然是直接的，我们大多数人是通过地图而不是通过旅游知道贵州的地理位置的。

  • B 归纳推理
  • C 拟真推理
  • D 回溯推理

  1. （x）演绎推理是指用一些特殊命题来证明一般性道理的命题。

  2. （x）反证法与选言证法均属于直接证明的范畴。
