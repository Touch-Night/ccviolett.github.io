Windows 有哪些好用的隐藏功能？

2022 年 2 月 25 日 阅读时间：1 分钟
ccviolett
Windows • WSL

一、自带截图

Win + Shift + S 即可唤出自带的截图软件。

二、自带 Linux 子系统

依次打开控制面板 -> Windows 功能 -> 适用于 Linux 的 Windows 子系统，勾选上，重启电脑。

随后在 Microsoft Store 中搜索想要安装的 Linux 子系统，如 Ubuntu，点击 获取，随后跟随指示完成即可。
